## 一、核心原理

### 1.1 什么是生成模型

**生成模型（Generative Model）** 是学习数据分布 p(x)，并能从该分布中采样生成新样本的模型。

```
判别模型 vs 生成模型：

判别模型（Discriminative）：
├─ 目标：学习 p(y|x)（给定x，预测y）
├─ 典型：SVM、Logistic Regression、CNN分类器
└─ 应用：分类、检测、分割

生成模型（Generative）：
├─ 目标：学习 p(x)（数据本身的分布）
├─ 能力：生成新样本、密度估计、缺失数据补全
└─ 典型：GAN、VAE、Diffusion、Flow
```

### 1.2 数学基础

#### 似然估计

生成模型的核心思想是**最大化数据似然**：

```python
# 给定数据集 {x_1, x_2, ..., x_N}
# 目标：找到参数 θ，使得数据出现的概率最大

max_θ Σ_{i=1}^N log p_θ(x_i)

# 其中 p_θ(x) 是模型建模的分布
```

#### 潜在空间（Latent Space）

```
高维数据空间（如图像 256×256×3）
    ↓ 编码器
低维潜在空间（如 z ∈ R^100）
    ↓ 解码器/生成器
重构/生成的高维数据

潜在空间的意义：
├─ 数据的压缩表示
├─ 连续、结构化的语义空间
└─ 可以进行插值、操作（改变风格、姿态等）
```

### 1.3 生成模型的分类

```
按似然是否可精确计算：

显式密度模型：
├─ 可精确计算：自回归模型（PixelCNN、GPT）
├─ 可精确计算：Normalizing Flows
└─ 可近似计算：VAE（ELBO下界）

隐式密度模型：
├─ 似然不可计算：GAN（通过采样学习）
└─ 似然可近似：Diffusion Model

按采样方式：
├─ 单步采样：GAN、VAE
├─ 多步迭代采样：Diffusion、PixelCNN
└─ 可逆采样：Normalizing Flows
```

## 三、GAN（生成对抗网络）

### 3.1 核心原理

**GAN** 通过**生成器（Generator）**和**判别器（Discriminator）**的对抗博弈来学习数据分布。

```
          噪声 z
           ↓
    ┌─────────────┐
    │  生成器 G   │ → 生成图像 G(z)
    └─────────────┘
           ↓
    ┌─────────────┐
    │  判别器 D   │ → 真假概率 D(x)
    └─────────────┘
           ↓
      真实图像 x
```

#### 损失函数（Minimax博弈）

```python
# 目标函数
min_G max_D V(D, G) = E_{x~p_data}[log D(x)] + E_{z~p_z}[log(1 - D(G(z)))]

# 判别器D：最大化区分真假的能力
# 生成器G：最小化判别器区分真假的能力

# 实际训练中，生成器常使用以下替代目标：
max_G E_{z~p_z}[log D(G(z))]  # 避免梯度消失
```

#### 纳什均衡

当训练收敛时：
- D(x) = 0.5 对所有x（判别器无法区分）
- p_g = p_data（生成分布 = 真实分布）
- 达到纳什均衡

### 3.2 主流变体

#### DCGAN（Deep Convolutional GAN）

```python
# 架构特点
class DCGAN_Generator(nn.Module):
    def __init__(self, latent_dim=100, ngf=64):
        super().__init__()
        self.main = nn.Sequential(
            # 输入: nz x 1 x 1
            nn.ConvTranspose2d(latent_dim, ngf * 8, 4, 1, 0, bias=False),
            nn.BatchNorm2d(ngf * 8),
            nn.ReLU(True),

            # 上采样: 4x4 -> 8x8 -> 16x16 -> 32x32 -> 64x64
            nn.ConvTranspose2d(ngf * 8, ngf * 4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 4),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf * 4, ngf * 2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 2),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf * 2, ngf, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf),
            nn.ReLU(True),

            nn.ConvTranspose2d(ngf, 3, 4, 2, 1, bias=False),
            nn.Tanh()  # 输出: [-1, 1]
        )

    def forward(self, z):
        return self.main(z)
```

**关键技巧**：
- 使用卷积层而非全连接
- 生成器用转置卷积上采样
- 判别器用步进卷积下采样
- BatchNorm稳定训练
- LeakyReLU用于判别器
- 避免池化层

#### StyleGAN系列

**StyleGAN/StyleGAN2/StyleGAN2-ADA/StyleGAN3**

```python
# StyleGAN核心思想：样式迁移
class StyleGenerator(nn.Module):
    """
    将潜在空间z → 中间空间w（Mapping Network）
    w控制每一层的样式（AdaIN）
    """
    def __init__(self):
        super().__init__()
        # 映射网络：8层MLP
        self.mapping = nn.Sequential(
            *[EqualizedLinear(512, 512) for _ in range(8)]
        )

        # 生成网络：由AdaIN控制风格
        self.generator = SynthesisNetwork()

    def forward(self, z):
        # z → w：解耦特征
        w = self.mapping(z)
        # 用w控制生成
        img = self.generator(w)
        return img


# AdaIN（自适应实例归一化）
def AdaIN(x, w):
    """
    用风格向量w调整特征x
    """
    return γ(w) * (x - μ(x)) / σ(x) + β(w)
```

**进展**：
- StyleGAN：引入样式控制
- StyleGAN2：改进归一化、渐进生长
- StyleGAN2-ADA：数据增强训练
- StyleGAN3：消除纹理粘连、更真实运动

#### CycleGAN

```python
# 无配对数据的图像转换
# 场景：马→斑马、夏天→冬天、素描→实景

class CycleGAN(nn.Module):
    """
    两个生成器 + 两个判别器 + 循环一致性损失
    """
    def __init__(self):
        self.G = Generator()  # A → B
        self.F = Generator()  # B → A
        self.D_A = Discriminator()  # 判断A的真假
        self.D_B = Discriminator()  # 判断B的真假

    def forward(self, x_a, x_b):
        # 前向循环
        fake_b = self.G(x_a)      # A → B
        recon_a = self.F(fake_b)  # B → A (重构)

        # 反向循环
        fake_a = self.F(x_b)      # B → A
        recon_b = self.G(fake_a)  # A → B (重构)

        return fake_b, recon_a, fake_a, recon_b


# 损失函数
def cycle_gan_loss(G, F, D_A, D_B, x_a, x_b):
    # 1. 对抗损失
    loss_G_GAN = adversarial_loss(D_B(G(x_a)), True)
    loss_F_GAN = adversarial_loss(D_A(F(x_b)), True)

    # 2. 循环一致性损失（关键！）
    loss_cycle = L1_loss(x_a, F(G(x_a))) + L1_loss(x_b, G(F(x_b)))

    # 3. 身份损失（保持颜色/结构）
    loss_id = L1_loss(x_a, F(x_a)) + L1_loss(x_b, G(x_b))

    return loss_G_GAN + loss_F_GAN + 10 * loss_cycle + 5 * loss_id
```

#### Pix2Pix / Pix2PixHD

```python
# 有配对数据的图像翻译
# 场景：语义图→真实图像、草图→照片、夜→昼

class Pix2Pix(nn.Module):
    """
    U-Net结构的条件GAN
    输入：条件图（如语义分割）
    输出：对应的目标图像

    def __init__(self, in_channels=3, out_channels=3):

class PatchGAN(nn.Module):
```

### 3.3 训练技巧

#### WGAN-GP（Wasserstein GAN with Gradient Penalty）

```python
# 解决原始GAN训练不稳定问题

class WGAN_GP(nn.Module):
    def __init__(self):
        super().__init__()
        self.generator = Generator()
        # 判别器输出标量（非概率）
        self.critic = Critic()

    def compute_gradient_penalty(self, real, fake):
        """
        梯度惩罚：限制判别器梯度范数≈1
        """
        # 随机插值
        alpha = torch.rand(real.size(0), 1, 1, 1, device=real.device)
        interpolates = (alpha * real + (1 - alpha) * fake).requires_grad_(True)

        # 计算判别器输出
        d_interpolates = self.critic(interpolates)

        # 计算梯度
        gradients = torch.autograd.grad(
            outputs=d_interpolates,
            inputs=interpolates,
            grad_outputs=torch.ones_like(d_interpolates),
            create_graph=True,
            retain_graph=True,
            only_inputs=True
        )[0]

        # 梯度惩罚
        gradient_penalty = ((gradients.norm(2, dim=1) - 1) ** 2).mean()
        return gradient_penalty

    def train_step(self, real_imgs):
        # 1. 训练判别器（n_critic次）
        for _ in range(5):
            z = torch.randn(real_imgs.size(0), latent_dim)
            fake_imgs = self.generator(z)

            # Wasserstein距离
            loss_critic = (
                self.critic(fake_imgs).mean() -
                self.critic(real_imgs).mean() +
                10 * self.compute_gradient_penalty(real_imgs, fake_imgs)
            )
            loss_critic.backward()
            critic_optimizer.step()

        # 2. 训练生成器
        z = torch.randn(real_imgs.size(0), latent_dim)
        fake_imgs = self.generator(z)
        loss_gen = -self.critic(fake_imgs).mean()
        loss_gen.backward()
        gen_optimizer.step()

        return loss_gen.item()
```

#### 谱归一化（Spectral Normalization）

```python
# 限制判别器Lipschitz常数

class SpectralNorm(nn.Module):
    """
    对权重矩阵W进行谱归一化：
    W_SN = W / σ(W)
    其中σ(W)是W的谱范数（最大奇异值）
    """
    def __init__(self, module, name='weight', power_iterations=1):
        super().__init__()
        self.module = module
        self.name = name
        self.power_iterations = power_iterations

        # 初始奇异向量
        w = getattr(module, name)
        height = w.data.shape[0]
        width = w.view(height, -1).data.shape[1]

        self.register_buffer('u', torch.randn(height))
        self.register_buffer('v', torch.randn(width))

    def forward(self, *args):
        # 幂迭代估计最大奇异值
        w = getattr(self.module, self.name)
        height = w.data.shape[0]

        # 归一化
        w_mat = w.view(height, -1)
        with torch.no_grad():
            for _ in range(self.power_iterations):
                self.v = F.normalize(
                    torch.mv(torch.t(w_mat), self.u), dim=0
                )
                self.u = F.normalize(
                    torch.mv(w_mat, self.v), dim=0
                )

        sigma = torch.dot(self.u, torch.mv(w_mat, self.v))
        setattr(self.module, self.name, w / sigma)

        return self.module.forward(*args)


# 使用
discriminator = nn.Sequential(
    SpectralNorm(nn.Conv2d(3, 64, 4, 2, 1)),
    nn.LeakyReLU(0.2),
    # ...
)
```

## 四、VAE（变分自编码器）

### 4.1 核心原理

**VAE** 通过变分推断学习数据的潜在表示，并从潜在空间生成新样本。

```
      输入 x
         ↓
    ┌─────────┐
    │ 编码器  │ → μ(x), σ(x)
    └─────────┘
         ↓
    采样 z ~ q(z|x) = N(μ, σ²I)
    （重参数化技巧）
         ↓
    ┌─────────┐
    │ 解码器  │ → 重构 x' = p(x|z)
    └─────────┘
```

#### ELBO（证据下界）

```python
# 目标：最大化 log p(x)
# 但 p(x) 不可计算，优化其下界 ELBO

log p(x) ≥ E_{z~q(z|x)}[log p(x|z)] - KL(q(z|x) || p(z))

#       = 重构损失         - KL散度

# 其中：
# p(x|z) = 解码器输出的分布
# q(z|x) = 编码器输出的近似后验
# p(z) = 先验（标准正态分布 N(0, I)）
```

### 4.2 重参数化技巧

```python
# 问题：从 q(z|x) = N(μ, σ²) 采样不可微分
# 解决：重参数化

class VAE(nn.Module):
    def encode(self, x):
        """
        编码器输出均值和方差
        """
        h = self.encoder(x)
        mu = self.fc_mu(h)      # [batch, latent_dim]
        log_var = self.fc_var(h)  # [batch, latent_dim]
        return mu, log_var

    def reparameterize(self, mu, log_var):
        """
        重参数化技巧
        z = μ + σ * ε,  ε ~ N(0, I)
        """
        std = torch.exp(0.5 * log_var)
        eps = torch.randn_like(std)  # 从标准正态采样
        return mu + eps * std  # 可微分的操作

    def decode(self, z):
        """
        解码器生成图像
        """
        return self.decoder(z)

    def forward(self, x):
        mu, log_var = self.encode(x)
        z = self.reparameterize(mu, log_var)
        x_recon = self.decode(z)
        return x_recon, mu, log_var


# 损失函数
def vae_loss(x_recon, x, mu, log_var):
    """
    重构损失 + KL散度
    """
    # 重构损失（假设伯努利分布）
    recon_loss = F.binary_cross_entropy(
        x_recon, x, reduction='sum'
    )

    # KL散度（解析解）
    # KL(N(μ, σ²) || N(0, 1)) = -0.5 * Σ(1 + log(σ²) - μ² - σ²)
    kl_loss = -0.5 * torch.sum(1 + log_var - mu.pow(2) - log_var.exp())

    return recon_loss + kl_loss
```

### 4.3 β-VAE

```python
# 通过调整β控制潜在空间的解耦程度

# 损失函数
L = E_q[log p(x|z)] - β * KL(q(z|x) || p(z))

# β > 1: 更强的KL约束，更解耦的潜在空间
# β < 1: 更关注重构，模糊的潜在空间

class BetaVAE(nn.Module):
    def __init__(self, beta=4.0):
        super().__init__()
        self.beta = beta
        # ... 其余架构同VAE

    def loss_function(self, x_recon, x, mu, log_var):
        recon_loss = F.binary_cross_entropy(x_recon, x, reduction='sum')
        kl_loss = -0.5 * torch.sum(1 + log_var - mu.pow(2) - log_var.exp())
        return recon_loss + self.beta * kl_loss
```

### 4.4 VAE变体

| 变体 | 特点 |
|:-----|:-----|
| **β-VAE** | 控制潜在空间解耦程度 |
| **VQ-VAE** | 离散潜在空间，向量量化 |
| **Conditional VAE** | 条件生成（CVAE） |
| **Adversarial Autoencoder** | 用对抗训练替代KL约束 |
| **NVAE** | 深层架构，流模型先验 |

#### VQ-VAE（Vector Quantized VAE）

```python
# 用于DALL-E、VQGAN的离散VAE

class VQVAE(nn.Module):
    """
    连续编码 → 离散码本 → 解码
    """
    def __init__(self, num_embeddings=512, embedding_dim=256):
        super().__init__()
        self.encoder = Encoder()  # 输出 [B, D, H, W]
        self.decoder = Decoder()

        # 码本：K个D维向量
        self.codebook = nn.Embedding(num_embeddings, embedding_dim)
        self.codebook.weight.data.uniform_(-1/num_embeddings, 1/num_embeddings)

    def forward(self, x):
        # 编码
        z_e = self.encoder(x)  # [B, D, H, W]

        # 向量量化
        # 1. 展平空间维度
        z_e_flat = z_e.permute(0, 2, 3, 1).contiguous()
        z_e_flat = z_e_flat.view(-1, z_e.shape[1])  # [B*H*W, D]

        # 2. 找最近码本向量
        distances = (
            torch.sum(z_e_flat ** 2, dim=1, keepdim=True)
            + torch.sum(self.codebook.weight ** 2, dim=1)
            - 2 * torch.matmul(z_e_flat, self.codebook.weight.t())
        )
        indices = torch.argmin(distances, dim=1)

        # 3. 量化
        z_q = self.codebook(indices)  # [B*H*W, D]
        z_q = z_q.view(z_e.shape[0], z_e.shape[2], z_e.shape[3], -1)
        z_q = z_q.permute(0, 3, 1, 2).contiguous()

        # Straight-through Estimator
        z_q_sg = z_e + (z_q - z_e).detach()  # 前向用z_q，反向用z_e

        # 解码
        x_recon = self.decoder(z_q_sg)

        return x_recon, z_e, z_q


# VQ-VAE损失
def vqvae_loss(x_recon, x, z_e, z_q, beta=0.25):
    """
    1. 重构损失
    2. Codebook Loss: 让码本向量靠近编码器输出
    3. Commitment Loss: 让编码器输出靠近码本向量
    """
    recon_loss = F.mse_loss(x_recon, x)

    # Codebook loss
    codebook_loss = F.mse_loss(z_q.detach(), z_e)

    # Commitment loss
    commitment_loss = F.mse_loss(z_q, z_e.detach())

    return recon_loss + codebook_loss + beta * commitment_loss
```

## 五、Diffusion Model（扩散模型）

### 5.1 核心原理

**扩散模型** 通过学习逐步去噪过程来生成数据，是目前最强大的生成模型之一。

```
前向过程（加噪）：
x_0 → x_1 → x_2 → ... → x_T (纯噪声)
每步加少量高斯噪声，T步后变为纯噪声

反向过程（去噪）：
x_T → x_{T-1} → ... → x_1 → x_0
训练神经网络学习每步的去噪过程
```

#### 前向扩散

```python
# 前向过程（Closed-form）
# q(x_t | x_{t-1}) = N(x_t; sqrt(1-β_t) * x_{t-1}, β_t I)

# 可直接采样任意t时刻：
# q(x_t | x_0) = N(x_t; sqrt(ᾱ_t) * x_0, (1-ᾱ_t) I)

# 其中：
# ᾱ_t = Π_{i=1}^t (1 - β_i)

def q_sample(x_0, t, noise=None):
    """
    从 x_0 直接采样 x_t
    """
    if noise is None:
        noise = torch.randn_like(x_0)

    sqrt_alpha_bar = ALPHA_BAR[t]**0.5
    sqrt_one_minus_alpha_bar = (1 - ALPHA_BAR[t])**0.5

    # x_t = sqrt(ᾱ_t) * x_0 + sqrt(1-ᾱ_t) * ε
    return sqrt_alpha_bar * x_0 + sqrt_one_minus_alpha_bar * noise
```

#### 反向去噪

```python
# 反向过程（神经网络学习）
# p_θ(x_{t-1} | x_t) = N(x_{t-1}; μ_θ(x_t, t), Σ_θ(x_t, t))

# 训练目标：预测噪声 ε
# L = E_{x_0, ε, t} [||ε - ε_θ(x_t, t)||²]

class DenoiseModel(nn.Module):
    """
    UNet架构预测噪声
    输入：带噪图像 x_t + 时间步 t
    输出：预测的噪声 ε_θ(x_t, t)
    """
    def __init__(self, in_channels=3, time_emb_dim=256):
        super().__init__()

        # 时间嵌入
        self.time_mlp = nn.Sequential(
            SinusoidalPositionEmbeddings(time_emb_dim),
            nn.Linear(time_emb_dim, time_emb_dim * 4),
            nn.GELU(),
            nn.Linear(time_emb_dim * 4, time_emb_dim)
        )

        # UNet架构
        self.conv_in = nn.Conv2d(in_channels, 64, 3, padding=1)

        # 下采样
        self.down1 = DownBlock(64, 128, time_emb_dim)
        self.down2 = DownBlock(128, 256, time_emb_dim)
        self.down3 = DownBlock(256, 512, time_emb_dim)

        # 中间层
        self.mid = nn.Sequential(
            ResBlock(512, 512, time_emb_dim),
            ResBlock(512, 512, time_emb_dim)
        )

        # 上采样
        self.up1 = UpBlock(512, 256, time_emb_dim)
        self.up2 = UpBlock(256, 128, time_emb_dim)
        self.up3 = UpBlock(128, 64, time_emb_dim)

        self.conv_out = nn.Sequential(
            nn.GroupNorm(32, 64),
            nn.SiLU(),
            nn.Conv2d(64, in_channels, 3, padding=1)
        )

    def forward(self, x, t):
        # 时间嵌入
        t_emb = self.time_mlp(t)

        # 下采样
        x1 = self.down1(self.conv_in(x), t_emb)
        x2 = self.down2(x1, t_emb)
        x3 = self.down3(x2, t_emb)

        # 中间
        x_mid = self.mid(x3)

        # 上采样（带跳跃连接）
        x = self.up1(x_mid, x3, t_emb)
        x = self.up2(x, x2, t_emb)
        x = self.up3(x, x1, t_emb)

        return self.conv_out(x)


# 训练循环
def train_diffusion(model, dataloader, num_timesteps=1000):
    # 预计算噪声参数
    beta = torch.linspace(0.0001, 0.02, num_timesteps)
    alpha = 1 - beta
    alpha_bar = torch.cumprod(alpha, dim=0)

    optimizer = torch.optim.Adam(model.parameters(), lr=1e-4)

    for epoch in range(num_epochs):
        for x_0, _ in dataloader:
            # 1. 随机采样时间步
            t = torch.randint(0, num_timesteps, (x_0.size(0),))

            # 2. 采样噪声
            noise = torch.randn_like(x_0)

            # 3. 加噪
            sqrt_alpha_bar_t = alpha_bar[t]**0.5[:, None, None, None]
            sqrt_one_minus_alpha_bar_t = (1 - alpha_bar[t])**0.5[:, None, None, None]
            x_t = sqrt_alpha_bar_t * x_0 + sqrt_one_minus_alpha_bar_t * noise

            # 4. 预测噪声
            predicted_noise = model(x_t, t)

            # 5. 计算损失
            loss = F.mse_loss(predicted_noise, noise)

            # 6. 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
```

#### 采样过程（DDPM）

```python
@torch.no_grad()
def ddpm_sample(model, num_samples, img_size, num_timesteps=1000):
    """
    DDPM采样：从纯噪声逐步去噪
    """
    model.eval()

    # 从纯噪声开始
    x = torch.randn(num_samples, 3, img_size, img_size)

    for t in reversed(range(num_timesteps)):
        # 预测噪声
        predicted_noise = model(x, t)

        # 计算均值
        alpha = 1 - beta[t]
        alpha_bar = alpha_bar[t]
        alpha_bar_prev = alpha_bar[t-1] if t > 0 else 1.0

        x = (
            1 / torch.sqrt(alpha) *
            (x - (1 - alpha) / torch.sqrt(1 - alpha_bar) * predicted_noise)
        )

        # 添加噪声（除了最后一步）
        if t > 0:
            noise = torch.randn_like(x)
            variance = (1 - alpha_bar_prev) / (1 - alpha_bar) * beta[t]
            x += torch.sqrt(variance) * noise

    return x.clamp(-1, 1)
```

### 5.2 DDIM（加速采样）

```python
@torch.no_grad()
def ddim_sample(model, x, t, t_prev, eta=0.0):
    """
    DDIM采样：非马尔可夫过程，更少采样步数

    eta=0: 确定性采样
    eta=1: DDPM采样
    """
    # 预测噪声
    predicted_noise = model(x, t)

    # 计算x_0预测
    alpha_t = alpha_bar[t]
    alpha_t_prev = alpha_bar[t_prev] if t_prev >= 0 else 1.0

    x_0_pred = (x - torch.sqrt(1 - alpha_t) * predicted_noise) / torch.sqrt(alpha_t)

    # DDIM更新
    sigma = eta * torch.sqrt((1 - alpha_t_prev) / (1 - alpha_t)) * torch.sqrt(1 - alpha_t / alpha_t_prev)

    x = torch.sqrt(alpha_t_prev) * x_0_pred + torch.sqrt(1 - alpha_t_prev - sigma**2) * predicted_noise

    if sigma > 0:
        x += sigma * torch.randn_like(x)

    return x


# DDIM采样循环
@torch.no_grad()
def ddim_sample_loop(model, num_samples, img_size, num_timesteps=1000, ddim_steps=50):
    model.eval()
    x = torch.randn(num_samples, 3, img_size, img_size)

    # 选择时间步子集
    timesteps = torch.linspace(num_timesteps - 1, 0, ddim_steps).long()

    for i in range(len(timesteps) - 1):
        t = timesteps[i]
        t_prev = timesteps[i + 1]
        x = ddim_sample(model, x, t, t_prev, eta=0.0)

    return x.clamp(-1, 1)
```

```python
# Stable Diffusion = VAE + UNet + 文本编码器

class StableDiffusion(nn.Module):
    """
    潜在扩散模型（LDM）
    在低维潜在空间（而非像素空间）进行扩散
    """
    def __init__(self):
        super().__init__()

        # 1. VAE: 像素空间 ↔ 潜在空间
        self.vae = AutoencoderKL(
            in_channels=3,
            out_channels=3,
            latent_channels=4,  # 潜在空间是4通道
            down_blocks=[1, 2, 4, 4],
            up_blocks=[4, 4, 2, 1]
        )

        # 2. UNet: 在潜在空间去噪
        self.unet = UNet2DConditionModel(
            sample_size=64,      # 潜在图像尺寸
            in_channels=4,
            out_channels=4,
            cross_attention_dim=768,  # 文本嵌入维度
            attention_head_dim=[5, 10, 20, 20]
        )

        # 3. 文本编码器: CLIP ViT-L/14
        self.text_encoder = CLIPTextModel.from_pretrained("openai/clip-vit-large-patch14")

        # 4. 采样器
        self.sampler = DDIMSampler()

    @torch.no_grad()
    def encode(self, images):
        """编码到潜在空间"""
        h = self.vae.encode(images).latent_dist.sample()
        return 0.18215 * h  # scaling factor

    @torch.no_grad()
    def decode(self, latents):
        """从潜在空间解码"""
        latents = 1 / 0.18215 * latents
        return self.vae.decode(latents).sample

    @torch.no_grad()
    def encode_prompt(self, prompt):
        """编码文本提示"""
        tokens = self.tokenizer(prompt)
        embeddings = self.text_encoder(tokens).last_hidden_state
        return embeddings

    @torch.no_grad()
    def generate(self, prompt, num_steps=50, guidance_scale=7.5):
        """
        生成图像

        Args:
            prompt: 文本提示
            num_steps: 采样步数
            guidance_scale: CFG强度（越高越符合提示）
        """
        # 1. 编码提示
        text_embeddings = self.encode_prompt(prompt)

        # 2. 无条件嵌入（用于Classifier-Free Guidance）
        uncond_embeddings = self.encode_prompt("")
        text_embeddings = torch.cat([uncond_embeddings, text_embeddings])

        # 3. 初始化潜在噪声
        latents = torch.randn(1, 4, 64, 64)

        # 4. DDIM采样
        for i, t in enumerate(self.timesteps):
            # 扩展latents用于CFG
            latent_model_input = torch.cat([latents] * 2)

            # UNet预测噪声
            noise_pred = self.unet(latent_model_input, t, encoder_hidden_states=text_embeddings)

            # Classifier-Free Guidance
            noise_pred_uncond, noise_pred_text = noise_pred.chunk(2)
            noise_pred = noise_pred_uncond + guidance_scale * (noise_pred_text - noise_pred_uncond)

            # DDIM更新
            latents = self.sampler.step(noise_pred, t, latents)

        # 5. 解码
        image = self.decode(latents)
        return image
```

## 六、Normalizing Flows

### 6.1 核心原理

**Normalizing Flows** 通过一系列可逆变换，将简单分布（如高斯）映射到复杂分布。

```python
# 基本思想
# z ~ p_z(z)  (简单分布，如标准正态)
# x = f(z)    (可逆变换)
# p_x(x) = p_z(f^-1(x)) |det(J_{f^-1}(x))|

# 其中：
# f^-1: 逆变换
# det(J): 雅可比行列式

# 如果变换序列为 f = f_K ∘ ... ∘ f_1
# p_x(x) = p_z(z) Π_{k=1}^K |det(d f_k / d z_k)|
```

### 6.2 RealNVP

```python
class RealNVP(nn.Module):
    """
    Real-valued Non-Volume Preserving变换
    使用仿射耦合层，雅可比矩阵可高效计算
    """
    def __init__(self, num_layers, num_hidden, data_dim):
        super().__init__()

        self.flows = nn.ModuleList()
        for i in range(num_layers):
            # 交替分割维度
            mask = torch.arange(data_dim) >= (data_dim // 2)
            mask = mask if i % 2 == 0 else ~mask

            self.flows.append(
                CouplingLayer(
                    num_hidden=num_hidden,
                    mask=mask
                )
            )

    def forward(self, x):
        """x → z，计算log_prob"""
        log_det = 0
        for flow in self.flows:
            x, ld = flow(x)
            log_det += ld

        # 假设z是标准正态
        log_prob = -0.5 * torch.sum(x**2, dim=1) - 0.5 * x.shape[1] * torch.log(torch.tensor(2 * torch.pi))
        log_prob += log_det

        return x, log_prob

    def inverse(self, z):
        """z → x，生成"""
        for flow in reversed(self.flows):
            z = flow.inverse(z)
        return z


class CouplingLayer(nn.Module):
    """
    仿射耦合层

    输入x分割为：
    - x1: d1 维（不直接变换）
    - x2: d2 维（用x1的仿射变换）

    前向：
    x1' = x1
    x2' = x2 * exp(s(x1)) + t(x1)

    反向：
    x1 = x1'
    x2 = (x2' - t(x1')) / exp(s(x1'))

    雅可比行列式 = exp(Σ s(x1))
    """
    def __init__(self, num_hidden, mask):
        super().__init__()
        self.mask = mask

        # s和t是MLP
        self.net = nn.Sequential(
            nn.Linear(mask.sum(), num_hidden),
            nn.ReLU(),
            nn.Linear(num_hidden, num_hidden),
            nn.ReLU(),
            nn.Linear(num_hidden, (~mask).sum() * 2)  # 输出s和t
        )

    def forward(self, x):
        """前向变换，返回y和log_det"""
        x1 = x[:, self.mask]
        x2 = x[:, ~self.mask]

        # 计算s和t
        st = self.net(x1)
        s, t = st.chunk(2, dim=1)

        # 仿射变换
        y1 = x1
        y2 = x2 * torch.exp(s) + t

        y = torch.cat([y1, y2], dim=1)

        # log_det = Σ s
        log_det = torch.sum(s, dim=1)

        return y, log_det

    def inverse(self, y):
        """反向变换"""
        y1 = y[:, self.mask]
        y2 = y[:, ~self.mask]

        st = self.net(y1)
        s, t = st.chunk(2, dim=1)

        x1 = y1
        x2 = (y2 - t) * torch.exp(-s)

        return torch.cat([x1, x2], dim=1)
```

```python
# Glow改进
class Glow(nn.Module):
    """
    Glow = Multi-scale架构 + 1x1卷积 + 仿射耦合
    """
    def __init__(self, num_channels, num_levels, num_steps_per_level):
        super().__init__()

        self.flows = nn.ModuleList()
        for level in range(num_levels):
            for step in range(num_steps_per_level):
                # 1. 1x1可逆卷积（替代Permutation）
                self.flows.append(Invertible1x1Conv(num_channels))

                # 2. 仿射耦合层
                self.flows.append(CouplingLayer(num_channels))

            # 3. 分割 squeeze（降低分辨率）
            self.flows.append(Squeeze())

    def forward(self, x):
        for flow in self.flows:
            x = flow(x)
        return x


class Invertible1x1Conv(nn.Module):
    """
    1x1可逆卷积
    用LU分解保证高效求逆和行列式计算
    """
    def __init__(self, num_channels):
        super().__init__()
        self.num_channels = num_channels

        # 随机初始化权重矩阵
        W = torch.randn(num_channels, num_channels)
        # LU分解
        P, L, U = torch.lu(W)

        # 可学习参数
        self.P = P  # 固定置换矩阵
        self.L = nn.Parameter(L)
        self.U = nn.Parameter(U)

        # 对角线初始化为1
        self.S = nn.Parameter(U.diag())

    def forward(self, x):
        """前向：y = W x"""
        W = self.P @ (self.L * (self.U + torch.diag(self.S - self.U.diag())))
        y = F.conv2d(x, W.view(self.num_channels, self.num_channels, 1, 1))
        log_det = torch.sum(torch.log(torch.abs(self.S)))
        return y, log_det

    def inverse(self, y):
        """反向：x = W^-1 y"""
        W = self.P @ (self.L * (self.U + torch.diag(self.S - self.U.diag())))
        W_inv = torch.inverse(W)
        x = F.conv2d(y, W_inv.view(self.num_channels, self.num_channels, 1, 1))
        return x
```

### 9.4 风格迁移：CycleGAN

```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader

class CycleGANTrainer:
    def __init__(self):
        # G: A→B, F: B→A
        self.G = ResNetGenerator(3, 3)  # A→B
        self.F = ResNetGenerator(3, 3)  # B→A

        # 判别器
        self.D_A = Discriminator(3)  # 判断A的真假
        self.D_B = Discriminator(3)  # 判断B的真假

        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.G.to(self.device)
        self.F.to(self.device)
        self.D_A.to(self.device)
        self.D_B.to(self.device)

    def train_step(self, real_a, real_b):
        """训练一个批次"""
        # 1. 生成假图像
        fake_b = self.G(real_a)      # A→B
        fake_a = self.F(real_b)      # B→A
        recon_a = self.F(fake_b)     # B→A→B
        recon_b = self.G(fake_a)     # A→B→A

        # 2. 判别器损失
        # D_A loss
        D_A_loss_real = self.gan_loss(self.D_A(real_a), True)
        D_A_loss_fake = self.gan_loss(self.D_A(fake_a.detach()), False)
        D_A_loss = (D_A_loss_real + D_A_loss_fake) * 0.5

        # D_B loss
        D_B_loss_real = self.gan_loss(self.D_B(real_b), True)
        D_B_loss_fake = self.gan_loss(self.D_B(fake_b.detach()), False)
        D_B_loss = (D_B_loss_real + D_B_loss_fake) * 0.5

        # 3. 生成器损失
        # 对抗损失
        G_gan_loss = self.gan_loss(self.D_B(fake_b), True)
        F_gan_loss = self.gan_loss(self.D_A(fake_a), True)

        # 循环一致性损失
        cycle_loss = (
            nn.L1Loss()(recon_a, real_a) +
            nn.L1Loss()(recon_b, real_b)
        ) * 10.0

        # 身份损失
        id_loss = (
            nn.L1Loss()(self.F(real_a), real_a) +
            nn.L1Loss()(self.G(real_b), real_b)
        ) * 5.0

        G_loss = G_gan_loss + F_gan_loss + cycle_loss + id_loss

        return {
            "G_loss": G_loss.item(),
            "D_A_loss": D_A_loss.item(),
            "D_B_loss": D_B_loss.item()
        }

    def gan_loss(self, pred, target_is_real):
        """GAN损失（LSGAN）"""
        if target_is_real:
            target = torch.ones_like(pred)
        else:
            target = torch.zeros_like(pred)
        return nn.MSELoss()(pred, target)
```

### 9.5 超分辨率：ESRGAN

```python
import torch
import torch.nn as nn

class RRDBNet(nn.Module):
    """
    Residual in Residual Dense Block for ESRGAN
    """
    def __init__(self, in_nc=3, out_nc=3, nf=64, nb=23):
        super().__init__()
        self.conv_first = nn.Conv2d(in_nc, nf, 3, 1, 1)

        # RRDB块
        self.RRDB_trunk = nn.Sequential(*[RRDB(nf) for _ in range(nb)])
        self.trunk_conv = nn.Conv2d(nf, nf, 3, 1, 1)

        # 上采样
        self.upconv1 = nn.Conv2d(nf, nf, 3, 1, 1)
        self.upconv2 = nn.Conv2d(nf, nf, 3, 1, 1)

        self.HRconv = nn.Conv2d(nf, out_nc, 3, 1, 1)

    def forward(self, x):
        fea = self.conv_first(x)
        trunk = self.trunk_conv(self.RRDB_trunk(fea))
        fea = fea + trunk

        # 上采样 x4
        fea = nn.functional.interpolate(
            fea, scale_factor=2, mode='nearest'
        )
        fea = self.upconv1(fea)

        fea = nn.functional.interpolate(
            fea, scale_factor=2, mode='nearest'
        )
        fea = self.upconv2(fea)

        out = self.HRconv(fea)
        return out


class RRDB(nn.Module):
    """Residual in Residual Dense Block"""
    def __init__(self, nf=64):
        super().__init__()
        self.conv1 = nn.Conv2d(nf, nf, 3, 1, 1)
        self.conv2 = nn.Conv2d(nf * 2, nf, 3, 1, 1)
        self.conv3 = nn.Conv2d(nf * 3, nf, 3, 1, 1)
        self.conv4 = nn.Conv2d(nf * 4, nf, 3, 1, 1)
        self.conv5 = nn.Conv2d(nf * 5, nf, 3, 1, 1)

    def forward(self, x):
        x1 = self.conv1(x)
        x2 = self.conv2(torch.cat([x, x1], dim=1))
        x3 = self.conv3(torch.cat([x, x1, x2], dim=1))
        x4 = self.conv4(torch.cat([x, x1, x2, x3], dim=1))
        x5 = self.conv5(torch.cat([x, x1, x2, x3, x4], dim=1))
        return x5 * 0.2 + x
```

```python
        # 分子表示：节点（原子）+ 边（化学键）
        self.node_embed = nn.Embedding(num_atom_types, hidden_dim)
        self.edge_embed = nn.Embedding(num_bond_types, hidden_dim)

        # 图去噪网络（类似UNet）
        self.denoise_net = MolecularDenoiseNet(
            node_dim=hidden_dim,
            edge_dim=hidden_dim,
            hidden_dim=hidden_dim,
            num_layers=6
        )

        self.num_steps = 1000

    def forward_diffusion(self, graph, t):
        """前向加噪"""
        # 为节点加噪
        node_noise = torch.randn_like(graph.x)
        node_alpha = self.alpha[t].view(-1, 1)
        graph.x_noisy = graph.x * node_alpha + node_noise * (1 - node_alpha)

        # 为边加噪
        edge_noise = torch.randn_like(graph.edge_attr)
        edge_alpha = self.alpha[t].view(-1, 1, 1)
        graph.edge_attr_noisy = graph.edge_attr * edge_alpha + edge_noise * (1 - edge_alpha)

        return graph.x_noisy, graph.edge_attr_noisy

    def sample(self, num_nodes, prompt=None):
        """采样新分子"""
        device = next(self.parameters()).device

        # 初始化噪声
        x = torch.randn(num_nodes, self.hidden_dim, device=device)
        edge_attr = torch.randn(num_nodes * 2, self.hidden_dim, device=device)

        # 逐步去噪
        for t in reversed(range(self.num_steps)):
            # 预测噪声
            pred_noise = self.denoise_net(x, edge_attr, t, prompt)

            # 去噪
            alpha = self.alpha[t]
            x = (x - pred_noise * (1 - alpha)) / alpha

        # 解码为分子
        atoms = self.decode_atoms(x)
        bonds = self.decode_bonds(edge_attr)

        return atoms, bonds

    def decode_atoms(self, x):
        """将节点嵌入解码为原子类型"""
        logits = self.atom_head(x)
        atom_types = torch.argmax(logits, dim=-1)
        return atom_types

    def decode_bonds(self, edge_attr):
        """将边嵌入解码为键类型"""
        logits = self.bond_head(edge_attr)
        bond_types = torch.argmax(logits, dim=-1)
        return bond_types


class MolecularDenoiseNet(nn.Module):
    """分子去噪网络"""
    def __init__(self, node_dim, edge_dim, hidden_dim, num_layers):
        super().__init__()

        # 时间嵌入
        self.time_mlp = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim * 4),
            nn.GELU(),
            nn.Linear(hidden_dim * 4, hidden_dim)
        )

        # 图注意力层
        self.layers = nn.ModuleList([
            MolecularGATLayer(node_dim, hidden_dim)
            for _ in range(num_layers)
        ])

        # 输出头
        self.node_head = nn.Linear(hidden_dim, node_dim)
        self.edge_head = nn.Linear(hidden_dim, edge_dim)

    def forward(self, x, edge_attr, t, prompt=None):
        t_emb = self.time_mlp(self.time_embed(t))

        # 图传播
        for layer in self.layers:
            x = layer(x, edge_attr, t_emb)

        # 预测噪声
        node_noise = self.node_head(x)
        edge_noise = self.edge_head(edge_attr)

        return node_noise, edge_noise

def graph_to_smiles(atoms, bonds, positions):
    """
    将分子图转换为SMILES字符串
    使用RDKit库
    """
    from rdkit import Chem
    from rdkit.Chem import AllChem

    mol = Chem.RWMol()

    # 添加原子
    for i, atom_type in enumerate(atoms):
        mol.AddAtom(Chem.Atom(atom_type.item()))

    # 添加键
    for i, (src, dst) in enumerate(zip(positions[::2], positions[1::2])):
        bond_type = bonds[i].item()
        mol.AddBond(src.item(), dst.item(), Chem.BondType(bond_type))

    # 生成SMILES
    smiles = Chem.MolToSmiles(mol)
    return smiles
```

```python
class MinibatchDiscrimination(nn.Module):
    """让判别器看整个批次"""
    def __init__(self, num_features, num_kernels):
        super().__init__()
        self.T = nn.Parameter(
            torch.randn(num_features, num_kernels * num_features)
        )

    def forward(self, x):
        # 计算批次内样本相似度
        M = torch.matmul(x, self.T)
        M = M.view(-1, num_kernels, x.size(1))

        # 计算每个样本与其他样本的距离
        o = torch.exp(-torch.abs(M.unsqueeze(1) - M.unsqueeze(2))).sum(2)
        return torch.cat([x, o], dim=1)
```

### Q3: 评估指标

**FID（Fréchet Inception Distance）**：
```python
from scipy import linalg
from torch.nn.functional import adaptive_avg_pool2d

def calculate_fid(real_images, generated_images, batch_size=32):
    """
    FID衡量生成图像与真实图像分布的距离
    越低越好
    """
    import torch
    from torchvision.models import inception_v3

    # 加载预训练Inception
    inception = inception_v3(pretrained=True, transform_input=False).eval()
    inception.fc = torch.nn.Identity()

    def get_features(images):
        features = []
        for i in range(0, len(images), batch_size):
            batch = images[i:i+batch_size]
            batch = adaptive_avg_pool2d(batch, (299, 299))
            feat = inception(batch)
            features.append(feat.detach())
        return torch.cat(features).cpu().numpy()

    # 获取特征
    real_features = get_features(real_images)
    gen_features = get_features(generated_images)

    # 计算均值和协方差
    mu1, sigma1 = real_features.mean(axis=0), np.cov(real_features, rowvar=False)
    mu2, sigma2 = gen_features.mean(axis=0), np.cov(gen_features, rowvar=False)

    # 计算FID
    ssdiff = np.sum((mu1 - mu2) ** 2)
    covmean = linalg.sqrtm(sigma1.dot(sigma2))

    if np.iscomplexobj(covmean):
        covmean = covmean.real

    fid = ssdiff + np.trace(sigma1 + sigma2 - 2 * covmean)
    return fid
```

**IS（Inception Score）**：
```python
def calculate_inception_score(images, batch_size=32, splits=10):
    """
    IS衡量生成图像的质量和多样性
    越高越好
    """
    from torchvision.models import inception_v3
    from scipy.stats import entropy

    inception = inception_v3(pretrained=True).eval()

    preds = []
    for i in range(0, len(images), batch_size):
        batch = images[i:i+batch_size]
        batch = adaptive_avg_pool2d(batch, (299, 299))
        pred = torch.nn.functional.softmax(inception(batch), dim=1)
        preds.append(pred.detach().cpu().numpy())

    preds = np.concatenate(preds)

    # 计算KL散度
    split_scores = []
    for k in range(splits):
        part = preds[k * (len(preds) // splits): (k + 1) * (len(preds) // splits), :]
        py = np.mean(part, axis=0)
        scores = []
        for i in range(part.shape[0]):
            pyx = part[i, :]
            scores.append(entropy(pyx, py))
        split_scores.append(np.exp(np.mean(scores)))

    return np.mean(split_scores), np.std(split_scores)
```
