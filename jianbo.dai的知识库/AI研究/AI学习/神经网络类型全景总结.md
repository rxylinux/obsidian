## 📊 按应用领域分类

### 1️⃣ 计算机视觉

| 网络类型 | 适用场景 | 代表模型 |
|:---------|:---------|:---------|
| **CNN** | 图像分类、目标检测 | ResNet, VGG, AlexNet |
| **ResNet** | 深层图像分类（解决退化） | ResNet-50, ResNet-101 |
| **VGG** | 特征提取、迁移学习基础 | VGG-16, VGG-19 |
| **Inception** | 高效图像分类 | GoogLeNet, Inception-v3 |
| **EfficientNet** | 轻量级移动端部署 | EfficientNet-B0~B7 |
| **MobileNet** | 移动端实时推理 | MobileNetV2, V3 |
| **ViT** (Vision Transformer) | 大规模图像预训练 | ViT-B/16, MAE |
| **U-Net** | 图像分割（医学图像） | U-Net, U-Net++ |
| **YOLO** | 实时目标检测 | YOLOv8, YOLOv10 |
| **Mask R-CNN** | 实例分割 | Mask R-CNN |

---

### 2️⃣ 自然语言处理

| 网络类型 | 适用场景 | 代表模型 |
|:---------|:---------|:---------|
| **RNN** | 短序列建模 | 简单RNN |
| **LSTM** | 长序列依赖（翻译、语音） | LSTM网络 |
| **GRU** | LSTM的轻量替代 | GRU网络 |
| **Transformer** | 通用NLP任务（当前主流） | BERT, GPT, T5 |
| **BERT** (Encoder-only) | 理解任务（分类、NER、QA） | BERT, RoBERTa |
| **GPT** (Decoder-only) | 生成任务（文本生成） | GPT-4, Llama 3, DeepSeek |
| **T5** (Enc-Dec) | 翻译、摘要 | T5, BART, FLAN-T5 |

> 想要系统学习 Transformer 架构？查看：[[AI研究/AI学习/02-模型原理/Transformer研读]]

---

### 3️⃣ 图结构数据

| 网络类型 | 适用场景 | 代表模型 |
|:---------|:---------|:---------|
| **GNN** | 图结构通用处理 | 各类GNN |
| **GCN** | 节点分类、链接预测 | Graph Convolutional Network |
| **GAT** | 需要不同权重的图注意力 | Graph Attention Network |
| **GraphSAGE** | 大规模图（采样聚合） | GraphSAGE |
| **GGNN** | 序列化推理、知识图谱 | Gated Graph Neural Network |

**应用场景**：
- 知识图谱：实体关系推理、链接预测
- 社交网络：好友推荐、社区发现
- 分子化学：药物发现、性质预测
- 交通网络：流量预测、路径规划

**实现框架**：
- PyTorch Geometric (PyG)
- DGL (Deep Graph Library)

---

### 4️⃣ 生成模型

| 网络类型 | 适用场景 | 代表模型 |
|:---------|:---------|:---------|
| **GAN** | 图像生成、风格迁移 | StyleGAN, CycleGAN |
| **VAE** | 数据生成、压缩、异常检测 | VAE, β-VAE |
| **Diffusion Model** | 高质量图像/视频生成 | Stable Diffusion, DALL-E 3, Sora |
| **Normalizing Flows** | 密度估计、生成 | RealNVP, Glow |

---

### 5️⃣ 序列与时间序列

| 网络类型 | 适用场景 | 特点 |
|:---------|:---------|:-----|
| **RNN/LSTM/GRU** | 传统序列建模 | 适合小规模数据 |
| **Transformer** | 大规模序列（当前主流） | 并行计算、长距离依赖 |
| **Mamba** | 长序列高效建模 | 线性复杂度（替代RNN） |
| **RWKV** | 长序列高效生成 | 线性注意力变体 |

---

### 6️⃣ 基础与专用网络

| 网络类型 | 适用场景 |
|:---------|:---------|
| **MLP** | 简单分类/回归、全连接层 |
| **Autoencoder** | 降维、特征学习、异常检测 |
| **Siamese Network** | 相似度比较（人脸验证、签名验证） |
| **Attention Mechanism** | 增强任何网络的聚焦能力 |
| **MoE** (Mixture of Experts) | 超大模型高效训练（Mixtral） |

## 📈 发展时间线

```
1958 感知机 (Perceptron)
  ↓
1986 MLP、反向传播 (Backpropagation)
  ↓
1998 LeNet（早期CNN）
  ↓
2012 AlexNet（CNN爆发）
  ↓
2014 VGG、GoogLeNet
  ↓
2015 ResNet（残差连接）
  ↓
2017 Transformer（"Attention is All You Need"）
  ↓
2018 BERT、GPT-1
  ↓
2020 GPT-3、ViT
  ↓
2022 Diffusion Model爆发（Stable Diffusion）
  ↓
2023-2024 LLM时代（GPT-4、Llama 3、DeepSeek）
  ↓
2025-? Mamba/RWKV等线性架构（探索中）
```

## 💡 当前的两大趋势

### 1. Transformer 统一江湖
- NLP：BERT、GPT、T5
- CV：ViT（Vision Transformer）、Swin Transformer
- 多模态：CLIP、Flamingo、GPT-4V

### 2. Diffusion Model 崛起
- 图像生成领域超越 GAN
- 视频生成（Sora）
- 3D 生成（Point-E）
- 多模态生成

## 🔗 核心概念辨析

> [!tip] 深入理解
> 想要更深入理解神经网络、架构、模型之间的层次关系？查看：[[AI研究/AI学习/2026-03-01]]

### 神经网络 vs 模型

| 概念 | 含义 | 例子 |
|:-----|:-----|:-----|
| **神经网络** | 架构类型/算法结构 | CNN、RNN、Transformer |
| **架构** | 具体的结构设计 | ResNet、LSTM、原始Transformer |
| **模型** | 训练好的具体实例 | ResNet-50、BERT-base、GPT-4 |

> [!example] 类比
> - **神经网络** = 建筑技术（大类）
> - **架构** = 设计图纸（具体方案）
> - **模型** = 建好的房子（实际产物）
> - **LLM** = 用 Transformer 结构建造的超大型建筑
