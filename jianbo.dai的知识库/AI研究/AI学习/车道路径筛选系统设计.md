# 车道路径筛选系统设计

> [!info] 需求说明
> 已知 A 到 B 的通行 link，需要细化到每个车道的可行驶路径
> - 考虑每个车道的转向限制
> - 考虑路径中的转向要求
> - 输出可通行的车道区间

---

## 核心问题

```
场景示例：
A link (4车道) ────▶ [路口1:直行] ────▶ [路口2:右转] ────▶ B link

A link 车道配置：
┌─────────────────────────────────┐
│ 车道1(最左)  车道2  车道3  车道4(最右)│
│  左转 only   直行   直行   直行+右转    │
└─────────────────────────────────┘

路径要求分析：
路口1需要直行 → 车道1❌(不能直行)
路口2需要右转 → 车道4✅(可以右转)

结果：
❌ 车道1: 无法行驶（不满足路口1）
❌ 车道2: 满足路口1，但路口2无法右转
❌ 车道3: 满足路口1，但路口2无法右转（除非先变道到车道4）
✅ 车道4: 可通行整个路径
```

## 数据结构

### 车道定义

```python
from dataclasses import dataclass
from typing import List, Set
from enum import Enum

class TurnDirection(Enum):
    """转向方向"""
    STRAIGHT = "直行"
    LEFT = "左转"
    RIGHT = "右转"
    U_TURN = "掉头"
    LEFT_STRAIGHT = "左直"
    RIGHT_STRAIGHT = "直右"
    ALL = "所有方向"

@dataclass
class Lane:
    """车道"""
    lane_id: str              # 车道ID
    link_id: str              # 所属link
    position: int             # 位置，0=最左
    allowed_turns: Set[TurnDirection]  # 允许的转向

    def can_make_turn(self, turn: TurnDirection) -> bool:
        """判断是否可以执行某个转向"""
        if turn in self.allowed_turns:
            return True
        if TurnDirection.ALL in self.allowed_turns:
            return True
        return False
```

### 路径定义

```python
@dataclass
class TurnInstruction:
    """转向指令"""
    intersection_id: str
    required_turn: TurnDirection

@dataclass
class Route:
    """路径"""
    route_id: str
    start_link: str
    end_link: str
    instructions: List[TurnInstruction]
```
