## 一、线性代数

### 1.1 标量、向量、矩阵、张量

```python
import numpy as np

# 标量 (Scalar)
scalar = 5

# 向量 (Vector) - 一维数组
vector = np.array([1, 2, 3])
print(vector.shape)  # (3,)

# 矩阵 (Matrix) - 二维数组
matrix = np.array([[1, 2, 3],
                   [4, 5, 6]])
print(matrix.shape)  # (2, 3)

# 张量 (Tensor) - 多维数组
tensor = np.array([[[1, 2], [3, 4]],
                   [[5, 6], [7, 8]]])
print(tensor.shape)  # (2, 2, 2)
```

**在 AI 中的意义**
- 标量：超参数（学习率）
- 向量：特征、权重
- 矩阵：数据集、权重矩阵
- 张量：图像 (C×H×W)、批量数据 (B×C×H×W)

### 1.2 向量运算

#### 点积 (Dot Product)

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

# 几何意义：|a|·|b|·cos(θ)
# 表示两个向量的"相似度"和"投影"
dot_product = np.dot(a, b)  # 1×4 + 2×5 + 3×6 = 32
```

**应用**：
- 计算相似度（余弦相似度）
- 神经网络的前向传播：`output = input @ weights`
- Attention 中的分数计算

#### 范数 (Norm)

```python
v = np.array([3, 4])

# L1 范数：曼哈顿距离
l1 = np.linalg.norm(v, ord=1)  # |3| + |4| = 7

# L2 范数：欧几里得距离
l2 = np.linalg.norm(v, ord=2)  # √(3² + 4²) = 5
```

**应用**：
- L1 正则化：产生稀疏解（特征选择）
- L2 正则化：防止过拟合（权重衰减）

### 1.3 矩阵运算

#### 矩阵乘法

```python
A = np.array([[1, 2],
              [3, 4]])  # (2, 2)
B = np.array([[5, 6],
              [7, 8]])  # (2, 2)

# 矩阵乘法：(m×n) @ (n×p) = (m×p)
C = A @ B  # 或 np.dot(A, B)
# [[19, 22],
#  [43, 50]]
```

**直观理解**：
- A 的行 × B 的列
- 每个结果元素是对应行的"线性组合"

**应用**：
- 神经网络层：`output = activation(input @ weights + bias)`

#### 转置 (Transpose)

```python
A = np.array([[1, 2, 3],
              [4, 5, 6]])  # (2, 3)

A_T = A.T  # (3, 2)
# [[1, 4],
#  [2, 5],
#  [3, 6]]
```

#### 逆矩阵 (Inverse)

```python
A = np.array([[1, 2],
              [3, 4]])

A_inv = np.linalg.inv(A)
# 方阵 A 的逆矩阵满足：A @ A_inv = I
```

> [!warning] 非方阵没有逆矩阵
> 但可以计算伪逆 (Pseudo-inverse)

### 1.4 特征值与特征向量

```python
A = np.array([[4, 1],
              [2, 3]])

eigenvalues, eigenvectors = np.linalg.eig(A)
# A @ v = λ · v
```

**直观理解**：特征向量是矩阵变换"方向不变"的向量

**应用**：
- PCA（主成分分析）
- PageRank 算法
- 图神经网络

## 二、微积分

### 2.1 导数

#### 定义

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

**直观理解**：函数在某点的"变化率"或"斜率"

#### 常见导数公式

| 函数 f(x) | 导数 f'(x) |
|:----------|:-----------|
| $x^n$ | $n \cdot x^{n-1}$ |
| $e^x$ | $e^x$ |
| $\ln(x)$ | $1/x$ |
| $\sin(x)$ | $\cos(x)$ |
| $\cos(x)$ | $-\sin(x)$ |

### 2.2 偏导数

多变量函数对其中一个变量的导数：

$$\frac{\partial f}{\partial x}$$

**例子**：
$$f(x, y) = x^2 + y^2$$
$$\frac{\partial f}{\partial x} = 2x, \quad \frac{\partial f}{\partial y} = 2y$$

### 2.3 梯度 (Gradient)

多元函数的偏导数组成的向量：

$$\nabla f = \left[\frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots, \frac{\partial f}{\partial x_n}\right]$$

**直观理解**：函数增长最快的方向

**应用**：梯度下降
```python
# 梯度下降更新参数
params = params - learning_rate * gradient
```

### 2.4 链式法则 (Chain Rule)

复合函数的导数：

$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$

**应用**：反向传播 (Backpropagation)

```python
# 简化的反向传播
# loss = f(g(h(x)))
# d(loss)/d(x) = f' · g' · h'
```

## 三、概率统计

### 3.1 随机变量与分布

#### 离散 vs 连续

| 类型 | 例子 | 概率质量/密度函数 |
|:-----|:-----|:-----------------|
| 离散 | 抛硬币 | PMF: P(X=x) |
| 连续 | 身高 | PDF: f(x) |

#### 常见分布

**伯努利分布**：单次试验（0/1）
$$P(X=x) = p^x (1-p)^{1-x}, \quad x \in \{0, 1\}$$

**二项分布**：n 次独立伯努利试验
$$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$$

**正态分布**（高斯分布）
$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

**应用**：
- 权重初始化：正态分布、Xavier 初始化
- Dropout：伯努利分布
- 贝叶斯推断：后验分布

### 3.2 期望与方差

**期望 (Expected Value)**：均值
$$E[X] = \sum x \cdot P(X=x) \quad (\text{离散})$$
$$E[X] = \int x \cdot f(x) dx \quad (\text{连续})$$

**方差 (Variance)**：离散程度
$$Var(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2$$

**标准差 (Standard Deviation)**：
$$\sigma = \sqrt{Var(X)}$$

### 3.3 贝叶斯定理

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

**直观理解**：后验 = 似然 × 先验 / 证据

**应用**：
- 朴素贝叶斯分类器
- 贝叶斯优化
- 变分推断

## 四、信息论

### 4.1 熵 (Entropy)

**信息量**：发生概率越小的事件，信息量越大
$$I(x) = -\log_2 P(x)$$

**熵**：平均信息量（不确定性）
$$H(X) = -\sum P(x) \log P(x)$$

**最大熵**：均匀分布时熵最大（最不确定）

**应用**：
- 决策树：用信息增益分裂
- 强化学习：最大化熵探索

### 4.2 交叉熵 (Cross-Entropy)

$$H(p, q) = -\sum p(x) \log q(x)$$

**直观理解**：用分布 q 来编码分布 p 的代价

**应用**：分类问题的损失函数
```python
# 交叉熵损失
loss = -sum(y_true * log(y_pred))
```

### 4.3 KL 散度 (Kullback-Leibler Divergence)

$$D_{KL}(p \| q) = \sum p(x) \log \frac{p(x)}{q(x)}$$

**直观理解**：两个分布的"距离"

**应用**：
- VAE 损失函数
- RL 中的策略优化
- 生成模型评估
