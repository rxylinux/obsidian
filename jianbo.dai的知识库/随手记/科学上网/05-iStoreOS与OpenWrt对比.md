## 常见问题

### Q1: iStoreOS 能用 OpenWrt 的插件吗？

```
✅ 是的！大部分 OpenWrt 插件都可以在 iStoreOS 上使用。

iStoreOS 的插件格式就是 OpenWrt 的 .ipk 格式。

如果你在 OpenWrt 论坛找到某个插件:
1. 下载 .ipk 文件
2. 在 iStoreOS 上通过 "系统→软件包→上传" 安装
3. 或放入 iStoreOS 的软件商店目录
```

### Q2: iStoreOS 是官方 OpenWrt 吗？

```
❌ 不是。iStoreOS 是基于 OpenWrt 的第三方发行版。

就像:
• 小米 MIUI ≠ Android (但基于 Android)
• iStoreOS ≠ OpenWrt (但基于 OpenWrt)

iStoreOS 由 linkease (小手冰凉) 团队维护。
```

### Q3: 我该选哪个？

```
新手用户 → 100% 推荐 iStoreOS
老手用户 → 看需求:
   • 想省心 → iStoreOS
   • 想折腾 → OpenWrt
   • 想定制 → OpenWrt
```

### Q4: iStoreOS 收费吗？

```
❌ 完全免费！iStoreOS 是开源项目，基于 GPL 协议。

不过:
• 有些高级插件可能需要捐赠才能下载
• 商业使用建议支持开发者
```

### Q5: 可以在 iStoreOS 上安装 OpenClash 吗？

```
✅ 可以！

iStoreOS = OpenWrt 22.03
OpenClash 支持 OpenWrt 21.03+

所以完全兼容！

安装方法见之前提供的教程。
```

## 总结

### 核心区别

| 方面 | OpenWrt | iStoreOS |
|------|---------|----------|
| **定位** | 原始系统 | 优化发行版 |
| **难度** | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| **界面** | 传统 | 现代化 |
| **商店** | ❌ | ✅ |
| **NAS** | ❌ | ✅ |
| **插件** | 全部 | 精选 |

### 一句话总结

```
OpenWrt  = 功能强大的原材料
iStoreOS = 基于 OpenWrt 精装而成的成品
```

**选择建议：**
- 新手 → **iStoreOS**
- 高手 → **OpenWrt** 或 **iStoreOS**
- 追求简单 → **iStoreOS**
- 追求极致定制 → **OpenWrt**
