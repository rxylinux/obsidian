## 详细对比

### iStoreOS ⭐推荐

**定位**: 人人会用的路由 + NAS 系统

| 特点 | 说明 |
|------|------|
| **基于 OpenWrt 22.03** | 继承稳定性和丰富生态 |
| **图形化界面** | 现代化 Web 界面，直观易用 |
| **应用商店** | 一键安装插件，像手机一样简单 |
| **NAS 功能** | 内置文件共享、媒体服务器 |
| **Docker 支持** | 轻松部署容器应用 |

#### 优势

```
✅ 开箱即用，新手友好
✅ 界面美观，操作简单
✅ 应用商店，一键安装
✅ 路由 + NAS 二合一
✅ 持续更新，社区活跃
```

#### 劣势

```
❌ 支持的硬件较少
❌ 高级功能有限
❌ 性能调优不如 OpenWrt
```

#### 适用场景

```
✅ 家庭网络中心
✅ 轻级 NAS 需求
✅ 智能家居网关
✅ 新手入门软路由
```

---

### OpenWrt

**定位**: 最强大的开源路由系统

| 特点 | 说明 |
|------|------|
| **7000+ 软件包** | 功能无限扩展 |
| **完全开源** | 社区活跃，更新频繁 |
| **高度可定制** | 可以深度定制一切 |
| **多架构支持** | x86、ARM、MIPS 等 |

#### 优势

```
✅ 功能最强大
✅ 社区最活跃
✅ 硬件支持最广
✅ 完全免费开源
✅ 文档资源丰富
```

#### 劣势

```
❌ 学习曲线陡峭
❌ 配置复杂
❌ 需要命令行操作
❌ 插件管理依赖 opkg
```

#### 适用场景

```
✅ 技术爱好者
✅ 需要深度定制
✅ 特殊硬件设备
✅ 企业级部署
```

---

### iKuaiOS (爱快)

**定位**: 国内商业路由系统，流控专家

| 特点 | 说明 |
|------|------|
| **流控功能** | 业界领先的流量控制 |
| **多 WAN 负载均衡** | 多线路智能切换 |
| **行为管理** | 员工上网行为管控 |
| **简单易用** | 图形化界面清晰 |

#### 优势

```
✅ 流控功能强大
✅ 多线路管理
✅ 行为管控完善
✅ 界面直观
✅ 稳定性好
```

#### 劣势

```
❌ 商业软件，部分功能付费
❌ 开放性不如 OpenWrt
❌ 插件生态一般
```

#### 适用场景

```
✅ 中小企业
✅ 网吧
✅ 酒店
✅ 需要流控的环境
```

---

**定位**: OpenWrt 国内优化版

| 特点 | 说明 |

| **保持兼容** | 与 OpenWrt 高度兼容 |

#### 优势

```
✅ 国内网络优化
✅ WiFi 6 表现好
✅ 兼容 OpenWrt
✅ 社区活跃
```

#### 适用场景

```
✅ 国内使用
✅ WiFi 6 路由器
✅ 需要稳定性
```

### X-Wrt

**定位**: 性能优化版 OpenWrt

| 特点 | 说明 |
|------|------|
| **性能优化** | 针对千兆宽带优化 |
| **界面友好** | 比 OpenWrt 更好用的界面 |
| **预装插件** | 常用插件预装 |

#### 适用场景

```
✅ 千兆宽带用户
✅ 追求性能
✅ 不想折腾太多
```

## OpenClash 安装

### iStoreOS 安装 OpenClash

```
1. 打开 iStoreOS 管理界面
2. 进入 "iStore 商店"
3. 搜索 "OpenClash"
4. 点击安装
5. 等待安装完成
6. 重启路由器
7. 在 "服务" 中找到 OpenClash
8. 配置订阅或节点
9. 启动服务
```

### OpenWrt 安装 OpenClash

```bash
# 1. 下载 OpenClash 安装包
# GitHub: https://github.com/vernesong/OpenClash

# 2. 通过 SSH 上传到路由器
scp luci-app-openclash_*.ipk root@192.168.1.1:/tmp/

# 3. SSH 登录路由器
ssh root@192.168.1.1

# 4. 安装
opkg install /tmp/luci-app-openclash_*.ipk

# 5. 重启 LuCI
/etc/init.d/uhttpd restart

# 6. 浏览器访问 LuCI 界面
# 在 "服务" 中找到 OpenClash
```

### 配置 OpenClash

```
1. 获取订阅链接 (从机场或自建服务器)
2. 在 OpenClash 中选择 "配置订阅"
3. 粘贴订阅链接
4. 点击 "更新"
5. 选择节点
6. 启动 OpenClash
7. 设置运行模式 (规则模式/全局模式)
```
