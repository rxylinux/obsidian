## 什么是客户端

### 核心概念

```
┌─────────────────────────────────────────────────────────────┐
│               科学上网的三个组成部分                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐              │
│  │ 服务端   │ ←→ │ 客户端   │ ←→ │  应用   │              │
│  │ (协议)   │    │ (工具)   │    │ (浏览器) │              │
│  └──────────┘    └──────────┘    └──────────┘              │
│      ↑              ↑              ↑                        │
│  VPS服务器      你电脑上的      你使用的                   │
│  运行协议       软件工具         软件                       │
│                                                             │
│  例如:           例如:           例如:                     │
│  Xray/V2Ray     Clash/Meta      浏览器/游戏                 │
│  Hysteria2      sing-box                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 客户端的作用

| 功能 | 说明 |
|------|------|
| **协议支持** | 解析并连接各种代理协议 |
| **分流规则** | 智能判断哪些流量走代理 |
| **节点管理** | 管理多个服务器节点 |
| **订阅更新** | 自动获取节点配置 |
| **流量控制** | TUN模式、系统代理等 |

## Clash 系列

### 家族关系

```
Clash (原版，已停更)
    │
    ├─→ Clash Meta (Mihomo) ←─── 推荐 ⭐⭐⭐⭐⭐
    │
    └─→ Clash Verge Rev (GUI客户端)
```

### Clash (原版)

| 项目 | 状态 | 说明 |
|------|------|------|
| **Clash** | ❌ 已停止维护 | 2020年停止更新 |
| **ClashX** | ⚠️ 仍在更新 | macOS 版，但内核老旧 |

### Clash Meta (Mihomo) ⭐推荐

**Clash Meta** 是 Clash 的增强版本，由 MetaCubeX 团队维护，是目前最活跃的 Clash 分支。

```
┌─────────────────────────────────────────────────────────────┐
│              Clash Meta vs Clash (原版)                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Clash (原版)         Clash Meta (Mihomo)                   │
│     │                     │                                │
│  ❌ 停止维护            ✅ 持续更新                          │
│  ❌ 协议老旧            ✅ 支持最新协议                       │
│  ❌ 功能受限            ✅ 功能增强                           │
│  ❌ 规则不完善          ✅ 完善的规则集                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### Clash Meta 支持的协议

| 协议 | Clash | Clash Meta |
|------|-------|------------|
| Shadowsocks | ✅ | ✅ |
| VMess | ✅ | ✅ |
| VLESS | ❌ | ✅ |
| Trojan | ❌ | ✅ |
| Hysteria | ❌ | ✅ |
| Hysteria2 | ❌ | ✅ |
| Reality | ❌ | ✅ |
| TUIC | ❌ | ✅ |
| SSH | ❌ | ✅ |

#### Clash Meta 新增特性

```
✅ 支持 Reality 协议
✅ 支持 Hysteria2
✅ 支持 TUIC (基于 QUIC)
✅ 更强的规则引擎

```

```
│  │                                                     │    │
```

#### 主要功能

| 功能 | 说明 |
|------|------|
| **TUN 模式** | 创建虚拟网卡，接管所有流量 |
| **系统代理** | 传统代理模式，仅代理支持代理的软件 |
| **规则模式** | 按规则智能分流 |
| **全局模式** | 所有流量走代理 |
| **订阅管理** | 一键添加/更新订阅 |
| **实时测速** | 显示节点延迟和速度 |
| **连接管理** | 查看活跃连接 |
| **日志查看** | 实时查看连接日志 |

## 其他客户端

### v2rayN (Windows)

```
特点:
• 老牌 Windows 客户端
• 支持 V2Ray/Xray 全系列协议
• 图形界面简洁
• 支持订阅

缺点:
• 界面较老旧
• 分流规则不如 Clash
```

### Qv2ray (跨平台)

```
特点:
• 基于 Qt 的跨平台客户端
• 插件系统
• 支持 V2Ray/Xray

缺点:
• 开发活跃度下降
• 配置较复杂
```

### Shadowrocket (iOS)

```
特点:
• iOS 上最强大的代理工具
• 支持几乎所有协议
• 规则功能强大
• 配置灵活

缺点:
• 需要美区 Apple ID
• 付费 ($2.99)
• 仅 iOS
```

### Surfboard (iOS)

```
特点:
• 轻量级 iOS 客户端
• 基于 Clash 内核
• 价格便宜

缺点:
• 功能不如 Shadowrocket
• 已停止更新
```

### OpenClash (OpenWrt 路由器)

**OpenClash** 是运行在 **OpenWrt 路由器**上的 Clash 插件，实现全家设备自动代理。

```
┌─────────────────────────────────────────────────────────────┐
│              OpenClash 工作模式                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                        ┌─────────────┐                      │
│                        │  OpenWrt    │                      │
│                        │  路由器      │                      │
│                        │  +OpenClash │                      │
│                        └──────┬──────┘                      │
│                               │                             │
│        ┌──────────────────────┼──────────────────────┐       │
│        │                      │                      │       │
│   ┌────▼────┐           ┌────▼────┐           ┌────▼────┐   │
│   │  手机    │           │  电脑   │           │ 电视/平板 │   │
│   │ 无需配置 │           │ 无需配置 │           │ 无需配置  │   │
│   └─────────┘           └─────────┘           └─────────┘   │
│                                                             │
│   所有设备自动走代理，无需单独配置！                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### OpenClash vs Clash Meta

| 特性 | Clash Meta | OpenClash |
|------|------------|-----------|
| **运行环境** | 电脑/手机 | **OpenWrt 路由器** |
| **使用方式** | 每台设备装客户端 | **路由器统一代理** |
| **覆盖范围** | 单台设备 | **全家所有设备** |
| **内核** | Mihomo | Mihomo (Clash Meta) |
| **支持 VLESS+REALITY** | ✅ | ✅ |

#### 优缺点

| 优点 | 缺点 |
|------|------|
| ✅ 全家设备自动代理 | ❌ 需要刷 OpenWrt |
| ✅ 设备无需安装软件 | ❌ 配置较复杂 |
| ✅ 支持所有 Clash Meta 协议 | ⚠️ 需要有一定技术基础 |
| ✅ 支持分流规则 | ⚠️ 路由器性能影响 |
| ✅ 支持 VLESS+REALITY | |

#### 适合人群

```
✅ 适合 OpenClash 的情况:
   • 家里有多个设备需要代理
   • 有技术能力刷 OpenWrt 固件
   • 想要一次配置，全家使用
   • 有性能较好的路由器 (建议 2GB+ 内存)

❌ 不适合:
   • 只有一台电脑或手机
   • 不想折腾路由器
   • 路由器性能较弱
   • 技术基础薄弱
```

#### 安装简要步骤

```bash
# 1. 准备支持的路由器并刷 OpenWrt
# 2. 下载 OpenClash 插件
# 3. 在 OpenWrt 中安装插件
# 4. 配置订阅或节点
# 5. 启动服务

# 下载地址:
# GitHub: https://github.com/vernesong/OpenClash
```

## 配置格式

### Clash 配置结构 (YAML)

```yaml
# Clash Meta 配置示例

# 混合端口 (HTTP + SOCKS5)
mixed-port: 7890

# 允许局域网连接
allow-lan: false

# 绑定 IP
bind-address: '*'

# 日志级别
log-level: info

# IPv6
ipv6: true

# 外部控制
external-controller: 127.0.0.1:9090

# 节点配置
proxies:
  - name: VLESS-Reality
    type: vless
    server: 1.2.3.4
    port: 443
    uuid: xxxx-xxxx-xxxx
    network: tcp
    tls: true
    udp: true
    flow: xtls-rprx-vision
    servername: www.cloudflare.com
    reality-opts:
      public-key: xxxx
      short-id: ""

  - name: Hysteria2
    type: hysteria2
    server: 1.2.3.4
    port: 443
    password: xxxx
    sni: example.com

# 代理组
proxy-groups:
  - name: Proxy
    type: select
    proxies:
      - VLESS-Reality
      - Hysteria2
      - DIRECT

  - name: Auto
    type: url-test
    proxies:
      - VLESS-Reality
      - Hysteria2
    url: 'http://www.gstatic.com/generate_204'
    interval: 300

  - name: Netflix
    type: select
    proxies:
      - VLESS-Reality
      - Hysteria2

# 分流规则
rules:
  # Netflix
  - DOMAIN-SUFFIX,netflix.com,Netflix
  - DOMAIN-SUFFIX,nflxvideo.net,Netflix

  # 国内直连
  - DOMAIN-SUFFIX,cn,DIRECT
  - GEOIP,CN,DIRECT

  # 广告拦截
  - DOMAIN-SUFFIX,ad.com,REJECT

  # 最终规则
  - MATCH,Proxy
```

## 总结

### 客户端选择建议

```
┌─────────────────────────────────────────────────────────────┐
│                      选择建议                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  新手用户:                                                  │
│    • Windows/macOS → Clash Verge Rev                       │
│    • Android → Clash Meta for Android                      │
│    • iOS → Shadowrocket (需付费)                           │
│                                                             │
│  家庭用户 (多设备):                                         │
│    • OpenWrt 路由器 + OpenClash                            │
│    • 一次配置，全家设备自动代理                             │
│                                                             │
│  进阶用户:                                                  │
│    • 尝试 sing-box 系列 (更灵活)                            │
```

### 如何选择？

| 场景 | 推荐方案 | 原因 |
|------|----------|------|
| 只用电脑 | **Clash Verge Rev** | 简单易用 |
| 只用手机 | **Clash Meta for Android / Shadowrocket** | 移动端优化 |
| 家里多个设备 | **OpenClash (路由器)** | 全家自动代理 |
| 不想折腾路由器 | **每台设备装客户端** | 上手简单 |
| 追求完美方案 | **路由器 + 客户端组合** | 灵活配置 |

### 核心要点

| 概念 | 说明 |
|------|------|
| **Clash** | 原版客户端，已停更 |
| **Clash Meta (Mihomo)** | 增强版，推荐使用 |
| **Clash Verge Rev** | Meta 的图形界面客户端 |
| **OpenClash** | OpenWrt 路由器版 Clash Meta |
| **协议** | 如 VLESS、Trojan 等 |
| **客户端** | 如 Clash、sing-box 等 |
| **区别** | 客户端支持多种协议 |

### 重要提醒

⚠️ **Clash Meta 完全支持 VLESS + REALITY 协议**

| 协议 | Clash 原版 | Clash Meta |
|------|-----------|------------|
| VLESS | ❌ | ✅ |
| REALITY | ❌ | ✅ |
| **VLESS + REALITY** | ❌ | ✅ |
| Hysteria2 | ❌ | ✅ |

这是推荐使用 Clash Meta (Mihomo) 的主要原因！
