# 科学上网实践记录

> [!info] 文档说明
> 记录从零开始搭建科学上网服务的完整过程

---

## 一、软硬件配置

**设备组合**: 中柏NS100 + 小米路由AX3600

- **中柏NS100**: 主路由
- **小米路由AX3600**: 通过有线连接，做无线路由中继
- **科学上网服务**: 使用 ssrdog
- **系统**: iStoreOS（基于OpenWrt）

---

## 二、安装iStore

> [!info] iStoreOS 简介
> iStoreOS是基于OpenWrt的增强版系统，提供图形化管理界面和丰富的插件支持

### 官方资源

| 资源类型     | 链接                                               | 说明                                 |
| -------- | ------------------------------------------------ | ---------------------------------- |
| **官方主页** | [www.istoreos.com](https://www.istoreos.com/)    | 系统的官方门户，包含功能介绍与导航                  |
| **固件下载** | [KoolCenter 酷友社](https://fw.koolcenter.com/)     | 官方存放各硬件型号固件（如 x86、R2S、R4S 等）的最直接地址 |
| **开源项目** | [GitHub - iStoreOS](https://github.com/istoreos) | 托管源码、文档 Wiki 以及问题反馈                |
| **交流论坛** | [酷友社论坛](https://koolcenter.com/)                 | 官方建议的交流平台，提供海量应用教程和问题解答            |
| **软件中心** | [iStore 仓库](https://github.com/linkease/istore)  | 存放 iStoreOS 内部插件中心所需的包文件           |

### 1. 准备工作

| 项目 | 说明 |
|-----|------|
| **固件下载** | 从 [iStoreOS官网](https://www.istoreos.com/) 下载对应型号的固件<br>x86通常选择带有 **x86_64** 字样的 `.img.gz` 文件<br>直链: [https://fw.koolcenter.com/iStoreOS/x86_64_efi/](https://fw.koolcenter.com/iStoreOS/x86_64_efi/) |
| **写盘工具** | 下载并安装 [Rufus](https://rufus.ie/) 或 [BalenaEtcher](https://www.balena.io/etcher/) |
| **硬件准备** | 准备一个 1GB 以上的 U 盘、显示器及键盘 |

### 2. 制作启动 U 盘

1. 插入 U 盘，打开 **Rufus**
2. 在"引导类型选择"中，点击"选择"并加载下载好的 iStoreOS 固件包（`.img.gz` 文件）
3. 点击"开始"将固件写入 U 盘
4. 等待写入完成

### 3. 安装系统

#### 1. 从 U 盘启动

1. 将 U 盘插入中柏NS100，连接显示器和键盘
2. 开机并从 U 盘启动
   - 重启并进入 BIOS/UEFI 设置（中柏NS100 按 **F7**）
   - 将 U 盘设为第一启动项

#### 2. 运行安装命令

1. 进入 iStoreOS 终端界面后，按下回车键
2. 输入以下安装命令：

   ```bash
   quickstart
   ```

3. 在弹出的菜单中选择 **Install X86**
4. 按照提示确认安装到机器的内置硬盘中

#### 3. 完成安装

安装完成后拔掉 U 盘，重启机器

## 三、主路由配置（中柏NS100）

### 1. 登录iStoreOS后台

电脑浏览器输入默认IP：`192.168.100.1`

默认密码一般为：`password`

### 2. 设置网络向导

进入系统后，点击首页的 **"网络向导"**

选择 **"配置为旁路由/主路由"** -> 选择 **"主路由"** 模式

### 3. 配置宽带拨号（WAN口）

根据光猫模式选择：

| 光猫模式 | 选择方式 |
|---------|---------|
| 桥接模式 | PPPoE拨号，输入宽带账号和密码 |
| 路由模式 | DHCP客户端（动态IP） |

### 4. 测试网络

配置完成后，检查iStoreOS首页，确保：
- WAN口已获取到IP地址
- 电脑可以正常上网

---

### 命令行备忘

用于快速配置网络IP地址

#### `ifconfig` - 查看当前设备地址

显示网络接口配置信息，查看当前IP地址

## 四、无线AP配置（小米路由AX3600）

> [!warning] 关键步骤
> 为了避免出现"双层NAT（两层局域网）"导致科学上网分流失效和网络变慢，必须关闭小米路由器的路由和DHCP功能，让它只做纯粹的无线发射器。

### 1. 进入小米路由器后台

电脑连接小米路由器的Wi-Fi或LAN口，浏览器输入 `192.168.31.1` 或 `miwifi.com` 进入后台。

### 2. 切换为AP模式

1. 点击顶部菜单的 **"常用设置"** -> **"上网设置"**
2. 页面拉到最下方，找到 **"工作模式切换"**
3. 选择 **"有线中继模式"**（部分固件叫AP模式），点击下一步并保存

### 3. 完成组网

此时小米路由器会自动重启。重启后：
- 所有连接小米Wi-Fi的手机和智能家居，都会直接从中柏NS100那里获取 `192.168.100.x` 的IP地址
- 所有网络流量的计算和分配都由中柏NS100强大的CPU来承担

## 五、安装OpenClash

> [!info] OpenClash 简介
> OpenClash 是一个基于 OpenWrt 的 Clash 客户端，用于实现科学上网功能

### 1. 下载 OpenClash

从 GitHub 下载最新版本的 OpenClash 插件：

[https://github.com/vernesong/OpenClash/releases](https://github.com/vernesong/OpenClash/releases)

下载对应架构的 `.ipk` 文件（x86 设备选择 `luci-app-openclash_..._all.ipk`）

### 2. 登录 iStoreOS 后台

浏览器输入 `192.168.1.2` 登录 iStoreOS 管理界面

### 3. 更新软件包

在 iStoreOS 命令行终端输入：

```bash
opkg update
```

### 4. 手动安装 OpenClash

1. 在左侧栏找到 **"iStore"** 或 **"系统"** -> **"软件包"**
2. 选择 **"手动上传"** 或 **"上传软件包"**
3. 上传步骤 1 下载的 OpenClash `.ipk` 文件
4. 点击安装，等待安装完成

### 5. 设置订阅地址

1. 左侧边栏选择 **"服务"** -> **"OpenClash"**
2. 首次进入会要求下载 **Meta Core**，选择一个延迟最低的节点下载
3. 点击 **"配置文件"** -> **"创建"** 或 **"增加"**
4. 粘贴从 ssrdog 拷贝的订阅地址
5. 保存并应用配置
